A plugin that give you control for show or hide widgets.

== Description ==
Widgets Controller gives you full control for show or hide widgets on WordPress Categories, Posts and Pages. It lets you dynamically show or hide widgets content by just a few mouse clicks. No knowledge of PHP required.
= Features =
* Widgets Controller display setting is supported for :
	- Home Page
	- 404 Page
	- Search Page
	- All Categories
	- All Posts
	- All Pages

== Installation ==
1. Upload `widgets-controller` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the Appearance -> Widgets
   You can show Widget Controller options below every Widget.
4. Edit the desired widgets.
