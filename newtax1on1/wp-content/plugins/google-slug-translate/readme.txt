将中文的文章别名翻译为英语。

== Description ==

这个插件会自动将中文的文章别名用 Google 翻译为英文。

== Installation ==

您可以在WordPress后台插件目录中搜索Google Slug Translate，安装并启用即可。无多余设置。

== Frequently Asked Questions ==

= 为什么启用后没有“Google翻译”按钮？ =

请确认您的永久链接结构中包含“%postname%”占位符。

== Screenshots ==

1. 输入文章标题，会自动将别名翻译为英文。
2. 您也可以点击“Google翻译”按钮手动翻译。
